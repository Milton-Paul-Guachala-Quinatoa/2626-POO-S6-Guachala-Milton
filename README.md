# Sistema de Gestión de Restaurante

## Estudiante

Nombre: Milton Paul Guachala Quinatoa

## Descripción

Este proyecto que realice es un sistema de restaurante hecho en Python. Permite registrar platillos y bebidas, mostrar los productos y cambiar el precio de un producto.

## Estructura del proyecto

- modelos/: Contiene las clases Producto, Platillo y Bebida.
- servicios/: Contiene la clase Restaurante.
- main.py: Ejecuta el programa y muestra el menú.
- README.md: Información del proyecto.

## Herencia

Las clases `Platillo` y `Bebida` heredan de la clase `Producto`.

## Encapsulación

Se utiliza el atributo privado `__precio`, que solo puede modificarse mediante un método.

## Polimorfismo

Se utiliza el método `mostrar_informacion()`, que muestra información diferente según si el objeto es un platillo o una bebida.

## Reflexión

Con este proyecto aprendí a organizar mejor el código usando Programación Orientada a Objetos. También comprendí cómo aplicar herencia, encapsulación y polimorfismo en un programa algo sencillo.
